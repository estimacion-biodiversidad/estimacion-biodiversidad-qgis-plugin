Para instalar el complemento:
1. Desempaque el archivo .ZIP descargado de este repositorio a través de la opción "Clone or download".
2. Coloque los archivos y carpetas desempacados en el directorio de complementos de QGIS 3 (ej. C:\Users\mvargas\AppData\Roaming\QGIS\QGIS3\profiles\default\python\plugins).
3. Active el complemento a través del menú de QGIS: "Plugins" -> "Manage and Install Plugins"
4. El complemento "Estimación de la Biodiversidad" debe estar disponible en el menú "Plugins" y también su ícono en la barra de herramientas de QGIS.
